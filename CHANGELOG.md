# Changelog

## [1.1.0] - 2026-04-09

### Added
- Right-click context menu on `.html` and `.htm` files (both in Explorer and editor)
- Quick Pick tool selector when using context menu
- Smart selection handling: selected text is always preferred over full file
- Clear modal pop-up for large files with clipboard instructions

### Improved
- Accurate small vs large file detection in both sidebar and context menu
- Better user messages and UX flow
- Updated README with correct functionality and installation instructions
- SEO-optimized package.json description and keywords

### Fixed
- Bug where selections in large files incorrectly triggered large-file mode

## [1.0.0] - 2026-04-08
